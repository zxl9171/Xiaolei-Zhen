An iOS remote controller of Yahoo! Web Player(fuzzy-ninja)

Implemented a remote controller for Yahoo! Web Player based on the iOS. In it, we used a chrome extension to fit all web pages that consist Yahoo! Web Player. Node.js and faye to construct the server, and QRcode to match the smart phone and the web page.

Investigation and realization of white-box cryptography

In traditional cryptography, we always assume that the operating environment (terminal) of the cryptographic algorithms is safe. But in daily application, however, the assumption is not always right. The password software implementation is always fully visible to the attackers. So we need to protect the key with some special method, to prevent the attackers from getting the key by observing the memory or executing the cryptographic software.

"Score Scratchcards", an extension of Chrome

After the School Registry website publish the score, the extension get it automatically and remind the user. The score is not directly presented to the user but in the form of scratch cards. The user can freely choose whether and when to scratch them, which can keep the score mysterious.

"Easy Course", a social network for information of classes

Due to the lack of a curriculum communication platform for college life, our team designed a website with ThinkPHP, JQuery and Bootstrap for curriculum communication. The website can automatically get the syllabus from the educational system of our school, according to which groups the users. Users can publish announcements, chat online and send messages to each other with SMS and email. GoogleCalendar is also supported to act as a reminder.
